# express-backend-template


### használat:

- **telepítés:**
    ```shell
    $ npm install
    ```
- **server elindítása:**
    ```shell
    $ npm start
    ```
port megváltoztatása:
- hozz létre egy `.env` file-t a project gyökerében.
- Írd a file-ba hogy: `PORT=4000` a *4000* helyén tetszőleges számmal.
